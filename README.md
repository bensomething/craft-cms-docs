# Craft CMS Docs

Search the official Craft CMS documentation, knowledge base, and glossary.

> This is an unofficial extension and is not affiliated with or endorsed by [Pixel & Tonic](https://pixelandtonic.com).

## Configuration

This extension uses required Raycast preferences. You will be prompted to set them when you first launch the extension, and can update them from the extension's preferences at any time. No API key or external account is required.

### Craft CMS Version

Choose which Craft CMS documentation version to use when searching the `CMS` and `All Docs` sections.

Available values:

- `5.x`
- `4.x`
- `3.x`
- `2.x`
- `1.x`

### Craft Commerce Version

Choose which Craft Commerce documentation version to use when searching the `Commerce` and `All Docs` sections.

Available values:

- `5.x`
- `4.x`
- `3.x`
- `2.x`
- `1.x`

### View Mode

Choose how search results are displayed:

- `Expanded`: Shows item details in Raycast's detail pane.
- `Compact`: Shows a denser list view without the detail pane.

## What the Preferences Affect

- `CMS` searches use the selected Craft CMS version.
- `Commerce` searches use the selected Craft Commerce version.
- `All Docs` mixes results from multiple sources, but CMS and Commerce results are still filtered using the versions above.
- `Cloud`, `Knowledge Base`, and `Glossary` are not versioned.

## Included Sources

The extension can search and browse:

- Craft CMS documentation
- Craft Commerce documentation
- Craft Cloud documentation
- Craft CMS Knowledge Base articles
- Craft CMS Glossary terms

It also supports bookmarks, items recently opened in the browser, and deeplinks between glossary terms.
